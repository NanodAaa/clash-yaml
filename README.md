# config.yaml
## 简介
自用 clash yaml 文件, 需要可以自行取用.  
使用双机场, 使用前在 proxy-providers 处粘贴订阅链接.  
为了 ip 稳定, 避免因 ip 改变而产生的风险, 默认将选择一个固定的节点. 可以自行设置需要固定转移的组.
